# 非遗文旅AI助手 🏮

基于 AI 大模型与 3D 数字交互技术，为游客提供定制化非遗文旅路线智能规划、360° 触控非遗文物观赏及 AR 脸谱体验，让千年非遗在指尖焕发新生。

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🧳 **AI 文旅规划** | DeepSeek 大模型对话，输入目的地即可生成非遗行程 |
| 🏺 **匠影工坊** | 3D 模型触控旋转展示非遗文物 |
| 🎭 **面谱绘心** | AR 摄像头实时脸谱虚拟试戴 |
| 👤 **个人中心** | 用户信息 & 分享传播 |

## 🛠 技术栈

- 微信小程序原生开发
- ModelScope API（DeepSeek-V4）
- CSS 3D Transform + 触控手势
- 国风美学设计（故宫红·靛蓝·鎏金）

## 🚀 快速开始

1. 克隆项目到本地
2. 用微信开发者工具打开
3. 在 `utils/config.js` 中配置 API Key
4. 编译运行

## ⚠️ 安全说明

`utils/config.js` 含 API Key，已加入 `.gitignore`，不会提交到仓库。
